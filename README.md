# 🛠️ mato_blog_helper v1.1

> **네이버 브랜드커넥트 / 쿠팡 / 마이리얼트립 / 공공데이터 API / 네이버 블로그** 분석·백업 통합 자동화 도구

---

## 📌 프로젝트 개요

`mato_blog_helper`는 블로그 콘텐츠 제작을 위한 **데스크톱 GUI 도구**입니다.  
다양한 플랫폼의 상품 정보, 이미지, 리뷰를 자동으로 수집·정리하고,  
AI 프롬프트 템플릿을 활용하여 블로그 원고를 쉽게 작성할 수 있도록 돕습니다.

---

## 🎯 주요 기능

### 1. 📦 통합 분석 모드
- 네이버 브랜드커넥트 / 쿠팡 / 마이리얼트립 URL을 **하나의 UI에서 동시 처리**
- URL에 따라 자동으로 플랫폼을 감지하여 적절한 분석기 실행
- TXT 파일로 키워드-URL 쌍을 일괄 업로드 가능

### 2. 🟢 네이버 브랜드커넥트 분석
- 네이버 스마트스토어 상품 페이지를 Selenium으로 크롤링
- **상품명, 가격(할인가/원가), 할인율, 평점, 리뷰수, 스토어명** 자동 추출
- 상품 이미지 고화질 다운로드 (최대 원본 화질)
- 리뷰 텍스트 최대 4건 수집

### 3. 🟠 쿠팡 분석
- 쿠팡 상품 HTML 소스를 직접 파싱 (TXT 파일로 업로드)
- **업체명, 상품가(일반/와우 회원가), 베스트 상품평** 추출
- 상품 썸네일 이미지 다운로드

### 4. 🔵 마이리얼트립 분석
- 마이리얼트립 상품 HTML을 파싱하거나 URL로 직접 크롤링
- **가격, 상품명, 평점, 후기수, AI 리뷰, 실 후기** 추출
- 상품 이미지 최대 10장 다운로드

### 5. 📝 네이버 블로그 (N블로그) 백업
- 네이버 블로그 게시글의 **본문 텍스트 + 이미지**를 통째로 백업
- **비로그인 모드** (자동) / **수동 로그인 모드** (로그인 후 크롤링) 지원
- 글자수(공백 제외), 키워드 사용 횟수 자동 분석
- 이미지 원본 화질 다운로드 (type=w2 전략)

### 6. 🌐 공공데이터 API 분석
- 한국 공공데이터포털 API와 연동하여 데이터 조회 및 다운로드
- 지원 API 목록:
  | API명 | 설명 |
  |--------|------|
  | 부산광역시 교육/강좌 정보 | 부산 지역 교육 강좌 목록 |
  | K-MOOC 강좌정보 | 국가평생교육 온라인 강좌 |
  | 한국관광공사 축제/행사정보 | 전국 축제·행사 일정 |
  | LH 분양임대공고 | LH 분양·임대 공고 (지역/상태 필터) |
  | 대한민국 공공서비스 정보 | 정부24 공공서비스 목록 |
- API Key 저장/관리 기능
- 게시판 형태로 데이터 조회 후 체크하여 다운로드

### 7. 📋 AI 프롬프트 관리
- `Prompt/` 폴더에 `.txt` 파일로 저장된 AI 프롬프트 템플릿 관리
- 드롭다운으로 선택 후 **전체 복사** 기능으로 바로 AI에 붙여넣기 가능
- 브랜드커넥트, 쿠팡, 마이리얼트립, 공공서비스, 해외(영어/일본어/중국어 등) 다양한 템플릿 제공

## 💻 설치 및 실행 방법

### 사전 요구사항

- **Python 3.9 이상**
- **Google Chrome** 브라우저 (Selenium 크롤링에 필요)

---

### 🪟 Windows

```powershell
# 1. 레포지토리 클론
git clone git@github.com:dbdp/automation_full_set.git
cd automation_full_set

# 2. 가상환경 생성 및 활성화
python -m venv .venv
.venv\Scripts\activate

# 3. 필수 패키지 설치
pip install -r requirements.txt

# 4. 실행
python main.py
```

### 🍎 macOS

```bash
# 1. 레포지토리 클론
git clone git@github.com:dbdp/automation_full_set.git
cd automation_full_set

# 2. 가상환경 생성 및 활성화
python3 -m venv .venv
source .venv/bin/activate

# 3. 필수 패키지 설치
pip install -r requirements.txt

# 4. 실행
python main.py
```

> **⚠️ macOS 참고사항**
> - `customtkinter`는 macOS에서도 정상 동작합니다.
> - Selenium + Chrome의 경우 `chromedriver`가 자동으로 관리됩니다 (selenium 4.6+ 내장).
> - 일부 `os.startfile()` 호출은 macOS에서 동작하지 않을 수 있습니다 (API 가이드 열기 등).

## 🚀 사용 방법

### 1. 프로그램 실행 후 상단 탭으로 모드 전환

| 탭 | 기능 |
|-----|------|
| **사용방법** | 프로그램 설명 및 공지사항 (Note.txt) |
| **프롬프트** | AI 프롬프트 템플릿 보기/복사 |
| **통합** | 네이버+쿠팡+마리트 통합 분석 |
| **네이버** | 네이버 브랜드커넥트 단독 분석 |
| **쿠팡** | 쿠팡 HTML 파싱 분석 |
| **마리트** | 마이리얼트립 분석 |
| **API** | 공공데이터 API 조회/다운로드 |
| **N블로그** | 네이버 블로그 글/이미지 백업 |

### 2. 키워드와 URL 입력

1. 키워드(분석 명칭)와 URL(분석 대상)을 행에 입력
2. `+ 행 추가` 버튼으로 여러 건 동시 작업 가능
3. `TXT 첨부` 버튼으로 키워드-URL 쌍이 담긴 파일 일괄 업로드

### 3. 결과 저장

- `전체 작업 시작` 클릭 후 저장 폴더 선택
- 결과물은 `선택한폴더/플랫폼명/키워드명/` 구조로 저장됨
- 각 폴더에 분석 텍스트(`.txt`) + 이미지 파일 생성

### 4. AI 블로그 원고 생성 (수동)

1. 분석 결과 텍스트를 확인
2. `프롬프트` 탭에서 적합한 템플릿 선택 → `전체복사`
3. AI 서비스(ChatGPT 등)에 프롬프트 + 분석 데이터 붙여넣기
4. 생성된 원고를 블로그에 게시

## ⚙️ 설정 파일

| 파일 | 역할 | 비고 |
|------|------|------|
| `api_settings.txt` | API Key 저장 (JSON) | 프로그램 내에서 자동 관리 |
| `api_list.txt` | 공공데이터 API 목록 | 이름-URL 쌍으로 직접 편집 가능 |
| `Note.txt` | 사용방법/공지사항 | 직접 수정 가능 |
| `Prompt/*.txt` | AI 프롬프트 템플릿 | 파일 추가/수정으로 관리 |

## ⚠️ 주의사항

- **저작권**: N블로그 백업 기능 사용 시, 타인의 제작물을 허가 없이 재가공·재사용하는 것은 저작권법에 위배되며 법적 책임은 사용자에게 있습니다.
- **API Key**: 공공데이터 API를 사용하려면 [공공데이터포털](https://www.data.go.kr/)에서 API Key를 발급받아야 합니다.
- **크롤링 제한**: 각 플랫폼의 이용약관을 준수하세요. 과도한 크롤링은 IP 차단 등의 문제를 야기할 수 있습니다.
